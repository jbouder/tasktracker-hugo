Welcome to the **TaskTracker** documentation.

TaskTracker is a full-stack monorepo:

- **Backend** — FastAPI, PostgreSQL, async SQLAlchemy 2, Alembic, managed with `uv`.
- **Frontend** — React 19, TypeScript, Vite, Tailwind v4, shadcn/ui, styled with the
  [Nebari design system](https://github.com/nebari-dev/nebari-design).
